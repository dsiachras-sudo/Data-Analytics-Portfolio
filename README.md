### üìÇ Projects in this Repository

#### Financial & Business Modeling (R)
* **`Censored_Data_Tobit_Model.R`**
    * **The Problem:** Standard regression fails when data is "capped" at zero (e.g., insurance claims, household spending).
    * **The Solution:** Implemented a Tobit Regression model to correctly analyze censored financial data without bias.

* **`Customer_Segmentation_Clustering.R`**
    * **The Problem:** Identifying distinct customer groups without predefined labels.
    * **The Solution:** Used K-Means clustering algorithms to segment customers based on income and spending behavior.

#### High-Performance Computing (Python)
* **`High_Speed_Data_Processing.py`**
    * **The Problem:** Processing large financial datasets (1M+ rows) in Excel is too slow.
    * **The Solution:** A high-speed NumPy script that uses vectorization to calculate Z-scores and Euclidean distance instantly, replacing slow loops.

#### Machine Learning & Algorithms (R)
* **`Text_Relevance_Classifier.R`**
    * **The Solution:** An NLP pipeline that automates document sorting (News vs. Ads) using Logistic Regression and Latent Semantic Analysis (LSA).

* **`Movie_Recommender_System.R`**
    * **The Solution:** A collaborative filtering engine that builds personalized recommendation lists based on user similarity scores.
